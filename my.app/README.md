# Verdant en Next.js

Esta carpeta contiene la app de ecommerce migrada a Next.js.

## Scripts

- `npm run dev`: inicia el entorno de desarrollo.
- `npm run build`: genera el build de produccion.
- `npm start`: levanta la app compilada.

## Deploy en Vercel

El repositorio tiene otros archivos en la raiz, por eso el deploy se configura para construir desde `my.app`.

- Install Command: `cd my.app && npm install`
- Build Command: `cd my.app && npm run build`
- Output: gestionado por Next.js automaticamente
